# Introdução 
Jogo de computador que permite simular o comportamento do sistema imunitário (SI). O jogador pode reforçar o sistema imunitário com vacinas. O SI é sujeito a agressões aleatórias que vão aumentando o nível de dificuldade. 

# Objectivo
O objectivo do jogador é coordenar o SI para defender o organismo das agressões. 

# Modo de jogo
Para aumentar de nível, o jogador tem de destruir as agressões a que o corpo vai sendo sujeito. Para isso tem de aumentar a produção de glóbulos brancos e coordenar a sua distribuição quando existem alertas de ameaça.

O jogador pode adquirir evoluções que lhe permitem derrotar as ameaças com mais facilidade. Essas evoluções podem ser permanentes, por exemplo vacinas, ou temporárias, ex.: vitaminas, antibióticos

É possível derrotar infecções bacterianas sem antibióticos, se existirem glóbulos brancos suficientes. Quando isso acontece, o SI melhora a sua capacidade para derrotar esse tipo de bactérias.

O sucesso do SI depende também da carga viral e do equilíbrio bacteriano. Outra forma de o jogador fortalecer o SI é tomando probióticos para equilibrar o tipo de bactérias existentes no organismo.
